技术要点：Vue3, Pinia, Vue-Router, Vue-Flow, BlockSuite, Vite, TypeScript
项目描述：自研全链路可视化搭建平台，集成数据源加工、页面拖拽组装、逻辑编排引擎及自动化出码模块，旨在通过无代码方式快速交付业务需求。


项目业绩:
页面组装模块设计，选用通用blocksuite方案，使页面元素及物料丰富度高度可定制
动态表单开发，借助Vee-validte,基于hook实现可维护更高的动态表单以及表单表单设计工具
物料出码功能，基于vue-json-preety 封装物料协议数据预览与编辑
基于Vue-Flow实现流程引擎编辑器，通过节点，边的抽象封装支持多种类型节点连接与流程处理
流程引擎及服务编排等内容实现，基于move事件封装vue-draggable组件实现组件拖拽与页面内容编排
数据源支持1000万行表格渲染与编辑，
